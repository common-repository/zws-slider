ZWS Slider is a simple and lightweight wordpress slider plugin

== Description ==

This plugin is a simple image slider for responsive wordpress website using wordpress core post type. You can easily add, remove and manage your slide images through wordpress admin panel. 

== Installation ==

1. Upload the entire `zws-slider` folder to the `/wp-content/plugins/` directory, or directly install the plugin through the 'Plugins' menu in WordPress.
2. A new plugin menu called `ZWS Slider` will appear upon activation of this plugin. `ZWS Slider` menu has two child menus `ZWS Slider` and `Add New` under it.
3. You can see all slides in `ZWS Slider` and use `Add New` to add new slides
5. [zwsslider] is the shortcode to display the ZWS Slider
